# Dev Notes — Morsl

## What this is

Morsl is a menu generator for Tandoor Recipes. It uses constraint-based linear programming (PuLP/CBC) to pick recipes matching user-defined criteria, serves them through a web UI, and optionally syncs orders back to Tandoor as meal plans.

## Architecture

- **FastAPI** backend with Pydantic Settings, singleton DI via `app/api/dependencies.py`
- **PuLP/CBC** solver in `solver.py` — hard/soft constraints, progressive relaxation
- **Vanilla JS + Alpine.js** frontend in `web/` — no build step
- **APScheduler** for cron-based generation in `services/scheduler_service.py`
- Data stored as JSON files in `data/` — no database

## Development

```bash
# Install with dev dependencies
pip install -e ".[dev]"

# Run tests
pytest

# Lint
ruff check .

# Start dev server
uvicorn morsl.app.main:app --reload --port 8321
```

## Key conventions

- **Python 3.12+** required
- **Atomic JSON writes** via `utils.atomic_write_json()` — use this, not raw `json.dump`
- **Profile names** validated by `_PROFILE_NAME_RE` regex in `config_service.py` — alphanumeric, spaces, hyphens, underscores only. This is a security boundary (prevents path traversal).
- **Admin auth** via `require_admin` dependency — checks X-Admin-Token header against TTL cache. When PIN is disabled, all requests pass through.
- **Credentials** from env vars take priority over `data/settings.json` (base64-encoded token)
- **Tests** in `tests/` — unit tests, no live Tandoor required. Integration tests marked `@pytest.mark.integration`.

## File layout

| Path | Purpose |
|------|---------|
| `morsl/app/main.py` | FastAPI app, lifespan, middleware, page routes |
| `morsl/app/api/routes/` | API endpoints grouped by domain |
| `morsl/app/api/dependencies.py` | Singleton services, admin auth |
| `morsl/services/` | Business logic (generation, orders, scheduling, etc.) |
| `morsl/solver.py` | LP-based recipe selection |
| `morsl/tandoor_api.py` | Tandoor HTTP client with caching and retry |
| `morsl/models.py` | Domain models (Recipe, Food, Keyword, Book) |
| `morsl/constants.py` | All magic numbers |
| `web/` | Frontend static files |
